# v1.1.0 (2026/07/02)

## 核心更新

本次发版包含较多新功能或增强优化，核心概览如下：
- **阅读模式**：支持高亮批注，支持阅读设置，支持阅读位置记忆&恢复
- **高亮批注管理页面**：支持搜索、筛选、聚合、排序
- **书签列表增强**：(内容较多，请看下方完整说明)
- **侧边栏标签模块增强**：新增树状模式，快速探索标签关联
- **搜索语法优化**：限定语法简化为 `field:keyword`、`field:"key1 key2"`
- **标签自动补全增强**：支持中文全拼和拼音首字母前缀匹配；支持 Enter 保存并退出编辑
- **自动检查更新**

完整说明如下：

## 阅读模式

- 支持高亮批注：
    - 高亮颜色包括黄色、绿色、蓝色、粉色、主题色
    - 可在阅读页面侧边栏-高亮tab进行删除、修改高亮颜色、复制
    - 高亮tab工具栏显示模式支持在隐藏、常驻显示间切换 
    - 通用设置页面-高亮可定义复制按钮功能
        - 复制按钮功能：复制高亮&批注，仅复制高亮，仅复制批注
        - 自定义复制格式
    - 防丢失：正文若重新解析，高亮仍会进行重定位恢复；无法重定位恢复的高亮&批注会明确提示，不会自动删除
- 支持阅读设置：
    - 主题：跟随全局，浅色，深色
    - 字体：系统默认，无衬线，衬线，等宽
    - 字号
    - 页面宽度：自动根据视口宽度调整，也可手动调整
    - 行高
- 支持阅读时间预估：可自定义阅读速度使得预估符合个人情况
- 支持阅读位置记忆，每次进入阅读模式会询问是否恢复，可通过自然滚动覆盖记忆位置
- 可在阅读页面侧边栏-详情tab进行书签数据的编辑（能力等同于书签详情查看弹窗）
- 阅读页面正文提取引擎切换为 Defuddle
- 阅读页面正文提取支持向 Defuddle 传递参数进行自定义正文提取元素、屏蔽元素等，也支持使用自定义 Python 脚本替代 Defuddle 引擎
- 阅读页面正文提取优先基于 HTML 快照，因此无需针对同一个网站写两套支持脚本，仅需提供 HTML 快照的自定义脚本即可

## 高亮批注管理页面

- 搜索：支持直接搜索高亮、批注包含的文字，也支持限定关键词`hl`、`ann`
- 筛选：按高亮批注的批注（有/无）、颜色、日期；按高亮批注所在书签的域名、标签
- 聚合：无/书签/域名/颜色
- 排序：创建时间的升序/降序，修改时间的升序/降序，随机
- 编辑：高亮、批注支持原地编辑
- 复制：通用设置页-高亮可自定义复制按钮功能、复制格式、多条分隔符
- 批量操作：删除高亮，清楚批注，更改颜色，复制高亮，复制批注，复制全部

## 书签列表增强

- 书签快捷编辑
    - 书签工具栏新增 4 个按钮，可以一键切换书签的标题、描述、标签、笔记到编辑状态，原地进行编辑
    - 移动端编辑态会在文本框外区域增加遮罩，点击遮罩即可保存&退出
    - 移动端编辑态会自动将所编辑书签滚动到顶部，退出编辑态恢复原本位置，避免键盘遮挡
- 书签快捷标签：实现一键为书签新增/删除指定的单个或多个标签
    - 每个快捷标签支持显示模式在文字、图标之间切换
    - 每个快捷标签的图标支持自定义，支持搜索面板（iconify）
    - 每个快捷标签支持直接显示在书签工具栏，或收纳在一个统一的快捷标签目录按钮里
- 书签工具栏增强
    - 书签工具栏目前包括书签日期、书签动作、快捷编辑、快捷标签、书签状态 5 个模块，支持自定义模块顺序，支持启用/禁用指定模块
    - 每个模块内部的按钮都支持自定义排序、启用/禁用
    - 书签动作、快捷编辑、书签状态支持显示模式在文字、图标之间切换
    - 书签日期支持自定义跳转目标：禁用，最新快照，阅读模式，互联网存档，高亮与批注
    - 所有动作改为 API 调用 + UI 乐观更新，消除全页面刷新
- 随机按钮增强：
    - 支持启用/禁用随机按钮
    - 支持多种随机模式：列表，单条目
    - 若随机模式为单条目，支持多种目标：URL，阅读页面，HTML 快照，书签详情
    - 修复随机按钮筛选条件丢失问题
- 批量编辑增强：
    - 支持跨页勾选
    - 新增阅读模式相关动作：创建 HTML 文章，删除所有文章
- 书签详情查看弹窗优化：
    - 优化加载速度
    - 支持书签数据原地编辑
    - 书签状态样式改为 chip
- Favicon 重构
    - 全局架构由书签级别改为域名级别，大幅减少网络请求
    - 新增多 provider 支持 + 退火重试 + 自动修复机制，提高 favicon 成功率
    - 新增批量动作“刷新 Favicon”，支持手动更新指定书签 Favicon
    - 支持随自定义域名归一化同步
- 标题：移除标题长度限制，避免由此带来的各类问题；优化了 Bookmarklet 对此的兼容
- 描述：由悬浮 tooltip 改为点击展开/收缩完整内容
- 笔记：
    - 限制最大显示行数为15行
    - 显示态/编辑态无感切换，自动保持滚动位置
    - 显示/隐藏机制重构，全局笔记状态与单个书签笔记状态独立控制
- 预览图：
    - 预览图占位图背景透明，减少视觉阻塞感
    - 加载失败时也显示预览图占位图，避免显示图片破损样式
    - 支持开启预览图时，关闭预览图占位图
- 侧边栏：
    - 用户数据模块新增高亮批注相关数据
    - 移除 drawer/teleport 机制，确保侧边栏数据源单一
    - 滚动条空间始终预留，避免内容变化时视觉抖动

## 侧边栏标签模块增强

新增树状模式，将关联标签以嵌套树状形式展示，按包含书签数降序排列，快速发现标签模式

## 搜索语法优化

- 简化限定语法为 `field:keyword`、`field:"key1 key2"`，移除原限定语法`field:(keyword)`
- 新增限定词：`hl` 为高亮，`ann` 为批注

## 标签自动补全增强

- 拼音支持
    - 全拼前缀匹配：输入`shujufenxi`或部分可匹配到`数据分析`
    - 拼音首字母前缀匹配：输入`sjfx`或部分可匹配到`数据分析`
- Enter 保存：当自动补全下拉框未显示时，回车键会保存当前标签并退出编辑态

## 自动检查更新

- 若距离上次检查更新大于等于24小时，会自动检查更新
- 若发现更新版本
    - 会在顶部导航栏“设置”按钮处显示小圆点
    - 会在“设置”菜单栏中显示“新版本”，点击可打开新版本弹窗
- 新版本弹窗
    - 直接显示新版本的发布日志
    - 可跳转到新版本的发布页
    - 点击“知道了”，可消除“设置”按钮处的小圆点

## 其他

- 新增：HTML 快照失败自动重试机制
- 优化：“已分享”页面导航逻辑
    - 无论当前登录用户是否开启分享，都显示“已分享”入口
    - 若当前登录用户未开启分享，跳转到所有人的分享页面
    - 若当前登录用户开启分享，但暂无分享书签，跳转到所有人的分享页面
    - 若当前用户开启分享，且有分享书签，跳转到当前登录用户的分享页
- 优化：限制 Bundle 拖拽区域至图标，避免移动端误触
- 修复：URL 空参数残留，模板硬编码`?`
- 重构：SVG icon 风格统一为 Tabler Icon，抽取到 `svg_icons.html` 进行统一管理
- 重构：统一 localStorage key 命名

- 🇨🇳 中文本地化：界面翻译；标签聚合支持中文；时区与日期处理；favicon服务默认使用query.domains，无需🪜

- 🎲 排序增强：新增随机排序；支持按删除时间排序（回收站中）
- 🤖 过滤器（Bundle）增强：新增支持二级过滤器；新增支持所有筛选项
- 🐞 元数据获取增强：默认解析支持更多主流网站；新增支持自定义脚本
